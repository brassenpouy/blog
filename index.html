事件概述

2026 年 3 月 24 日，Arm 在旧金山 Fort Mason 举办 "Arm Everywhere" CEO keynote，CEO Rene Haas 亲自发布 Arm AGI CPU——Arm 成立 35 年来首款自研量产数据中心处理器。这不再是 IP 授权或 CSS（Compute Subsystem）交付，而是 Arm 直接设计、由 TSMC 代工、以完整芯片形式销售的产品。

发布当日 ARM 股价飙升约 16%，创一年内最佳单日涨幅。Barclays 上调目标价至 $200，Evercore ISI 上调至 $227，Needham 从 Hold 升级为 Buy。

架构设计哲学

EVP Mohamed Awad 明确表示 AGI CPU 是一款 "clean sheet design"，刻意不添加对遗留应用的支持，将全部晶体管资源集中于 AI 工作负载：

  • 单线程/核：不采用 SMT，每核一个线程，确保 deterministic performance scaling。这与 NVIDIA Vera 形成差异化——Vera 支持 SMT
  • 内存与 I/O 集成：内存控制器和 I/O 集成到与计算核心相同的 die 上以最小化延迟，每个 socket 暴露为两个 NUMA domain
  • 不含加速器：避免集成"不会被 100% 利用"的功能，聚焦纯 CPU 编排能力

NVIDIA Vera 的 2 倍+

性能声明

Arm 宣称在机架级部署中 AGI CPU 比最新 x86 系统性能提升超过 2 倍：

  1. 更高的有效线程密度：x86 在持续负载下核心争抢致性能退化，Arm 的内存带宽优势保持更多有效线程
  2. 更高的单线程效率：Neoverse V3 每线程完成更多工作
  3. 复合效应：更多可用线程 × 更高单线程性能 = 每机架大幅领先

关键洞察
  • vs NVIDIA Vera：两者都基于 Arm 架构，在 standalone CPU 机架场景直接竞争。但 NVIDIA 拥有 GPU + NVLink + CUDA 全栈优势，Vera 更多作为 GPU 集群的伴侣 CPU
  • vs x86：Arm 的主要攻击目标。Arm 已占云数据中心 CPU 市场约 40%，AGI CPU 进一步加速蚕食
  • vs NVIDIA GPU：互补为主。Jensen Huang 在 GTC 表示 CPU 正成为新瓶颈，并发来祝贺视频
授权商冲突风险

Arm 从 IP 供应商转为芯片销售商，与自身授权客户形成"亦友亦敌"关系。DIGITIMES 报道 TSMC 投资的 GUC（创意电子）等将受冲击。但主要授权商均已表态支持——Arm 提供更多"可选性"而非替代，做大生态蛋糕对所有参与者有利。
