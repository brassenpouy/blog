事件概述

2026 年 3 月 24 日，Arm 在旧金山 Fort Mason 举办 "Arm Everywhere" CEO keynote，CEO Rene Haas 亲自发布 Arm AGI CPU——Arm 成立 35 年来首款自研量产数据中心处理器。这不再是 IP 授权或 CSS（Compute Subsystem）交付，而是 Arm 直接设计、由 TSMC 代工、以完整芯片形式销售的产品。

发布当日 ARM 股价飙升约 16%，创一年内最佳单日涨幅。Barclays 上调目标价至 $200，Evercore ISI 上调至 $227，Needham 从 Hold 升级为 Buy。

架构设计哲学

EVP Mohamed Awad 表示 AGI CPU 是 "clean sheet design"，全部晶体管资源集中于 AI 工作负载：

◆ 单线程/核 — 不采用 SMT，确保确定性性能扩展（与 NVIDIA Vera 的 SMT 形成差异化）

◆ 内存与 I/O 集成 — 集成到计算核心同一 die 上，最小化延迟

◆ 不含加速器 — 避免集成非 100% 利用的功能，聚焦纯 CPU 编排

性能声明

Arm 宣称机架级部署比最新 x86 系统性能提升超过 2 倍：

1. 更高有效线程密度 — x86 在持续负载下核心争抢致性能退化

2. 更高单线程效率 — Neoverse V3 每线程完成更多工作

3. 复合效应 — 更多线程 × 更高效率 = 每机架大幅领先

关键洞察

◆ vs NVIDIA Vera — 同为 Arm 架构，standalone CPU 场景直接竞争，但 NVIDIA 有 GPU+NVLink+CUDA 全栈优势

◆ vs x86 — Arm 的主攻目标，已占云 CPU 约 40%，AGI CPU 加速蚕食

◆ vs NVIDIA GPU — 互补为主。Jensen Huang 在 GTC 表示 CPU 正成为新瓶颈

授权商冲突风险

Arm 从 IP 供应商转为芯片销售商，与授权客户形成"亦友亦敌"关系。但主要授权商均表态支持——做大生态蛋糕对所有参与者有利。
